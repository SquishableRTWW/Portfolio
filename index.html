PHYSICS:

In this module, I learnt how to apply the physics concepts learnt throughout my secondary education into game development. Using a simple c++ openGL framework, I implemented many different physics concepts into my project, which was a 2D top-down mini-golf game.

Some of the concepts applied include: Friction,   elastic and inelastic collision,   gravity,   and   raycasting.

   









2D GAME CREATION:

In this module, I used a c++ framework to make a side-view 2D game. I implemented many game creation concepts, including level design, 2D game logic, AI behaviour and pathfinding. My project being shown is a dungeon escape game, where the player utilises protals and keys to escape, while being attacked by enemies.
